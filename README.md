# 🎁 Stardew Valley — Loved Gifts Guide

An interactive gift guide for **Stardew Valley** focused on easily obtainable loved gifts — crops, forageables, and animal products. No fruit trees.

**[Live Demo →](https://YOUR_USERNAME.github.io/stardew-gifts/)**

## Features

- **Seasonal gift guide** — crops and forageables organized by season
- **Animal products section** — Goat Cheese, Duck Feather, Void Egg, Wool, Truffle, Rabbit's Foot
- **Hard-to-gift characters** — researched easiest options (cooking, mining, artisan)
- **Character tracker** — check off maxed characters, they disappear from the gift list
- **Persistent progress** — uses localStorage, survives page refresh and revisits
- **Mobile-friendly** — works great on phone while playing

## Data Source

Based on Stardew Valley v1.6 wiki data. Excludes fruit tree items.

## Deploy Your Own

1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → `main` / `/ (root)`
4. Your guide will be live at `https://YOUR_USERNAME.github.io/stardew-gifts/`

## Single File

The entire app is a single `index.html` — React via CDN, no build step needed.
